# Computer Vision Fundamentals

This repository contains study notes and hands-on practice codes
from an AI Computer Vision fundamentals course.

Topics include:
- Image processing and convolution
- Edge detection (Sobel, Canny)
- Morphological operations
- Superpixel segmentation (SLIC)
- Local features
- Perceptron and basic deep learning concepts

All experiments are implemented using Python and Google Colab.
